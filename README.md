# security

HyperBEAM device project generated by `rebar3 new device`.

## Build

```sh
rebar3 compile
```

## Package

```sh
rebar3 device package
rebar3 device verify
```

## Test

```sh
rebar3 device test
rebar3 eunit-all
```

## Local Node

```sh
rebar3 device local
```

## Publish

```sh
rebar3 device publish --key wallet.json
```
